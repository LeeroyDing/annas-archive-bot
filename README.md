# annas-archive-bot

How to run:

1. Create your bot at @BotFather
2. Copy the access token
3. Export your token as environment variable:  
   Bash:  
   `export TOKEN='your token'`

   Powershell:  
   `$env:TOKEN = 'your token'`

4. Run the code:  
   `go get`  
   `go run .`
